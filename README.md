# dsasnippets README

## Features

Currently supports the following snippets (more to come):

#### Sorting
- [x] Selection ```$slctn-srt```

- [x] Insertion ```$ins-srt```

- [x] Bubble ```$bbl-srt```

- [x] Shell ```$shell-srt```

- [x] Merge ```$mrg-srt```

- [x] Quick ```$qk-srt```

- [x] Heap ```$hp-srt```

- [x] Counting ```$cnt-srt```

- [x] Radix ```$rx-srt```

#### Searching
- [x] Linear ```$lin-srch```

- [x] Binary ```$bi-srch```

- [x] Jump ```$jmp-srch```

- [x] Interpolation ```$interpl-srch```

- [x] Fibonacci ```$fib-srch```
